# Hawa Tasks Flutter App

Hawa is a Flutter task management application built with a clean UI and organized project structure.  
The app focuses on helping users manage their daily tasks in a simple and smooth way.

## Features

- Clean and modern Flutter UI
- Task management screens
- Organized project structure
- Custom app assets and splash logo
- Android and iOS support
- Simple and readable code structure

## Tech Stack

- Flutter
- Dart
- Material Design
- Android / iOS
